# データセット選定

## 背景

NYC Taxi Trip Data（`s3://nyc-tlc/`）が公開アクセス不可のため、代替の公開S3データセットを検討する。

---

## アクセス検証結果

| # | データセット | S3バケット | アクセス可否 | 備考 |
|---|-------------|-----------|-------------|------|
| 1 | Amazon Customer Reviews | `s3://amazon-reviews-pds/` | ❌ 不可 | 2023〜2024年頃に公開停止 |
| 2 | NOAA GSOD（気象データ） | `s3://noaa-gsod-pds/` | ✅ 可 | |
| 3 | COVID-19 Data Lake | `s3://covid19-lake/` | ✅ 可 | |
| 4 | GDELT（グローバルイベント） | `s3://gdelt-open-data/` | ✅ 可 | |
| 5 | OpenAQ（大気質） | `s3://openaq-fetches/` | ❌ 不可 | |

---

## アクセス可能な候補詳細

### 2. NOAA Global Surface Summary of Day（気象データ）
- **S3:** `s3://noaa-gsod-pds/`
- **形式:** CSV
- **規模:** 1億件以上（観測所×日次）
- **特徴:** 観測所・地点・日付ディメンションあり。時系列分析の練習向き
- **認証:** 不要

### 3. COVID-19 Data Lake（AWSキュレーション）
- **S3:** `s3://covid19-lake/`
- **形式:** Parquet / CSV
- **規模:** 中規模（国・州別日次データ）
- **特徴:** 複数データソースをAWSが整形済み。地域・日付ディメンションが明確
- **認証:** 不要

### 4. GDELT（グローバルイベントデータベース）
- **S3:** `s3://gdelt-open-data/events/`
- **形式:** CSV
- **規模:** 10億件以上（毎日更新）
- **特徴:** ニュースイベント・国・アクター・日付を軸にした大規模分析向き
- **認証:** 不要
